## 🚀 Probar la aplicación

**Demo en línea:** https://configurowebmax.github.io/python-recordatorios-pagos-servicios/

No requiere instalación ni registro: la app corre **100% en tu navegador** con Python (PyScript).

## ✨ Características

- 🐍 **Python real** ejecutándose en el navegador (PyScript / WebAssembly)
- 🔒 **Sin servidor, sin base de datos**: tus datos se guardan solo en tu navegador (localStorage)
- ⚡ **100% gratis** y de código abierto
- 📱 **Responsive**: funciona en móvil, tablet y escritorio
- 🌐 **Sin dependencias locales**: solo abrir el `index.html`

## 🛠️ Tecnología

- **Python** — toda la lógica de negocio (`app.py`)
- **PyScript** — runtime de Python en el navegador
- **HTML5 + CSS3** — interfaz
- **localStorage** — persistencia local
